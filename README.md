## API Documentation

### 1. Chats
#### `POST /chats`
- **Description**: Creates a new chat for an application.
- **Response**:
  - **201 Created**: Chat created.
  - **422 Unprocessable Entity**: Validation failed.


### 2. Messages
#### `POST /messages`
- **Description**: Creates a new message in a chat.
- **Response**:
  - **201 Created**: Message created.
  - **422 Unprocessable Entity**: Validation failed.
